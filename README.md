# Rustacean Theme

A dark VS Code theme inspired by Rust's mascot Ferris the crab, crafted for developers who appreciate the beauty of Rust programming.

## Features

- **Dark theme** optimized for reduced eye strain during long coding sessions
- **Rust-focused syntax highlighting** with special attention to Rust-specific constructs
- **Comprehensive language support** for JavaScript, TypeScript, Python, C++, and many more
- **Semantic highlighting** support for enhanced code understanding
- **Carefully chosen colors** that provide excellent contrast and readability

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Rustacean"
4. Click Install

Alternatively, install via the command line:

```bash
code --install-extension rustacean
```

## Usage

1. Open the Command Palette (Ctrl+Shift+P)
2. Type "Theme" and select "Preferences: Color Theme"
3. Choose "Rustacean" from the list

## Theme Colors

The theme features a carefully selected color palette:

- **Background**: Deep, warm brown (`#171203`) for a comfortable coding experience
- **Foreground**: Soft gray (`#5f5a4d`) that's easy on the eyes
- **Accent**: Rust-inspired orange (`#f8bf34`) and red (`#e03a20`) highlights
- **Syntax**: Comprehensive color scheme with distinct colors for different code elements

### Language-Specific Highlights

- **Rust**: Special highlighting for lifetimes, macros, and Rust-specific keywords
- **JavaScript/TypeScript**: Enhanced readability for modern web development
- **Python**: Clear distinction between keywords, strings, and variables
- **C/C++**: Proper highlighting for preprocessor directives and type modifiers
